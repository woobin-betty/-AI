# Task Manager App (AI 과제 관리)

이 저장소는 간단한 Vite + React + Tailwind 프로젝트입니다. `TaskManagerApp` 컴포넌트가 포함되어 있고, 로컬에서 실행하여 확인할 수 있습니다.

## 실행 방법

1. 프로젝트 폴더로 이동:
   ```bash
   cd task-manager-app
   ```
2. 의존성 설치:
   ```bash
   npm install
   ```
3. 개발 서버 실행:
   ```bash
   npm run dev
   ```
4. 브라우저에서 http://localhost:5173 을 엽니다.

## 깃허브에 업로드
압축파일을 풀고(또는 zip 파일의 디렉토리를 그대로) 로컬에서 `git init` 후 원격 저장소를 추가하고 `git push` 하면 됩니다.

```bash
git init
git add .
git commit -m "Initial commit - Task Manager App"
git branch -M main
git remote add origin <YOUR_GITHUB_REPO_URL>
git push -u origin main
```
